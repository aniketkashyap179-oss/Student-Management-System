Help & Support
Everything you need to use EduTrack effectively.

Adding Students

Go to Add Student in the sidebar. Fill all required fields (marked with *) and click Enroll Student. Data is saved automatically to the Excel sheet.

Viewing & Searching

Go to View Students to see all enrolled students. Use the search bar to filter by name, roll number, course, or any field in real-time.

Deleting Students

In View Students, click the red Delete button next to a student. A confirmation dialog will appear before permanent deletion.

Editing Records

Click the Edit button next to any student to open the edit modal. Modify the fields you want and click Save Changes.

Frequently Asked Questions
Where is the student data stored?
All student data is saved in an Excel file (students.xlsx) in the same directory as the application. This file can be opened with Microsoft Excel or Google Sheets.
Can I have duplicate roll numbers?
No. Roll numbers must be unique. The system will show an error if you try to add a student with an existing roll number.
How do I start the application?
Install the requirements with pip install -r requirements.txt, then run python main.py. Open your browser and visit http://localhost:8000.
Can I export student data?
Yes! Simply locate the students.xlsx file in your project directory. It's a standard Excel file you can share or import anywhere.
What happens if I delete a student?
Deletion is permanent. The student record is removed from the Excel file immediately. Always confirm before deleting.
